# Stories Index

A curated sequence for reading and validating the chat experience. Starts with first-run discovery, then core messaging flows, followed by advanced/maintenance stories.

1. [discovery.md](./discovery.md) - First launch, name modal, invite flow (QR/deep link)
2. [responding.md](./responding.md) - Recipient flow accepting an invitation and starting a chat
3. [managing-connections.md](./managing-connections.md) - View/select connections, filter/sort, create invites when needed
4. [sending-media.md](./sending-media.md) - Share photos, videos, documents, voice messages within chat
5. [searching-messages.md](./searching-messages.md) - Search across/all or within a single conversation; filters and media search
6. [editing-messages.md](./editing-messages.md) - Edit/delete sent messages; edit history and constraints
7. [deleting-channels.md](./deleting-channels.md) - Remove/archive channels; blocking and undo considerations
8. [video-call.md](./video-call.md) - Voice/video call initiation, accept/decline, switch modes, screen share
9. [profile-management.md](./profile-management.md) - Add/change/remove profile photo; shows in lists and bubbles

Notes:
- This order is the single source of truth for scenario ordering.
- Files intentionally excluded from indices: `AGENTS.md`, `README.md`, `STATUS.md`, and `index.md`.
